# GitHub Profile Template

This template provides a structured approach to maintaining your GitHub profile README.

## Template Variables

You can customize the following variables in `resources/profile-config.json`:

- `profile.name` - Your display name
- `profile.username` - GitHub username
- `profile.email` - Contact email
- `social.linkedin` - LinkedIn profile URL
- `technologies.*` - Your tech stack
- `projects[]` - Featured projects
- `theme.*` - Visual theme settings

## Customization Tips

1. **Update Personal Info**: Edit the profile section in `profile-config.json`
2. **Add New Technologies**: Update the technologies object with new skills
3. **Feature New Projects**: Add projects to the projects array
4. **Change Theme**: Modify theme settings for different color schemes
5. **Add Custom Badges**: Create new badge URLs in `resources/README.md`

## Maintenance

- Update project links regularly
- Keep technology list current
- Check badge URLs are working
- Monitor GitHub stats API limits
